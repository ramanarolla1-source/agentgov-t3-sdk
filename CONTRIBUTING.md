Contributing to AgentGov
Thank you for your interest in AgentGov! We are building the trust foundation for enterprise AI, bridging autonomous agents with cryptographically verified corporate governance.

Current Project Status
AgentGov is currently an MVP under active development for the Terminal 3 Agent Dev Kit Developer Challenge.

Accepting Contributions: At this stage, I am not accepting external Pull Requests as I am focusing on core architectural stability for the challenge submission.

Feedback & Issues: I highly encourage feedback, bug reports, and discussions regarding the T3 SDK integration or governance logic. Please use the GitHub Issues tab to start a conversation.

How to Engage
If you find a security vulnerability, an architectural flaw, or have a suggestion for improving the trust-stack integration, please:

Open an Issue: Describe the context, the behavior observed, and your suggested improvement.

Contact: Reach out via my professional channels if you are interested in technical collaboration or enterprise deployment pilots.

Code of Conduct
By participating in this project, you agree to maintain a professional, constructive, and respectful environment. This project is built to support secure and accountable enterprise operations—all discourse should reflect those values.
